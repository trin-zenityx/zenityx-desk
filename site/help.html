เริ่มเชื่อมต่อสิทธิ์บน MacWindowsคำถามที่พบบ่อย

เริ่มต้น: เชื่อมต่อ 1:1

ZenityX Desk ให้ "ครู" เชื่อมเข้าไปคุมจอ "นักเรียน" ได้ — ทั้งสองฝ่ายเปิดแอปเดียวกัน

  1. นักเรียนเปิดแอป จะเห็น ID (เลข 9 หลัก) และ รหัสผ่านครั้งเดียว ที่แถบซ้าย
  2. นักเรียนส่ง ID + รหัสให้ครู
  3. ครูพิมพ์ ID นักเรียนในช่อง Control Remote Desktop → กด Connect → ใส่รหัสผ่าน
  4. ครูจะเห็นและควบคุมหน้าจอนักเรียนได้ทันที
💡 สำหรับนักเรียนประจำ: ตั้ง รหัสผ่านถาวร ได้ที่ไอคอนดินสอข้างรหัสผ่าน จะได้ไม่ต้องส่งรหัสใหม่ทุกครั้ง

🍎 ให้สิทธิ์บน macOS

ครั้งแรกที่ให้ครูคุมเครื่อง macOS ต้องอนุญาต 2 อย่าง (ทำครั้งเดียว):

1. Screen Recording (เห็นหน้าจอ)

  1. System Settings → Privacy & Security → Screen Recording
  2. เปิดสวิตช์ให้ ZenityX Desk

2. Accessibility (ควบคุมเมาส์/คีย์บอร์ด)

  1. Privacy & Security → Accessibility → เปิดให้ ZenityX Desk
⚠️ สำคัญ: หลังเปิด Screen Recording ต้อง ปิดแล้วเปิดแอปใหม่ ถึงจะมีผล (ถ้าไม่ทำ แอปจะเด้งขอสิทธิ์ซ้ำ)

เปิดแอปครั้งแรกเจอ "unidentified developer"? → คลิกขวา ที่แอป → Open → ยืนยัน (ครั้งแรกครั้งเดียว)

🪟 Windows

  1. แตกไฟล์ .zip ลงโฟลเดอร์ แล้วเปิด rustdesk.exe
  2. ถ้า SmartScreen เตือน: คลิก More info → Run anyway
  3. แอปเชื่อมเซิร์ฟเวอร์ ZenityX อัตโนมัติ พร้อมใช้งาน
คำเตือนเกิดเพราะไฟล์ยังไม่ได้เซ็นใบรับรอง (กำลังดำเนินการ) — ดาวน์โหลดจาก desk.zenityx.com เท่านั้นเพื่อความปลอดภัย

คำถามที่พบบ่อย

เชื่อมต่อไม่ติด / ค้างที่ "connecting"

เช็กว่าทั้งสองเครื่องออนไลน์ (แถบล่างขึ้น "● Ready") และพิมพ์ ID ครบ 9 หลักถูกต้อง ลองปิด-เปิดแอปทั้งสองฝั่ง

เด้งขอสิทธิ์ Screen Recording ซ้ำๆ

หลังกดอนุญาตแล้วต้อง quit แล้วเปิดแอปใหม่ — เป็นข้อกำหนดของ macOS
