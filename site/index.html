🍎 macOS

1เปิด .dmg แล้วลาก ZenityX Desk ลง Applications
2เปิดครั้งแรก: คลิกขวา ที่แอป → Open → ยืนยัน
3เปิด Screen Recording + Accessibility ใน System Settings → Privacy
4ปิดแล้วเปิดแอปใหม่ — พร้อมเชื่อมต่อ
ถ้าขึ้น "unidentified developer" ใช้คลิกขวา → Open ครั้งแรกเท่านั้น

🪟 Windows

1แตกไฟล์ .zip ลงโฟลเดอร์ที่ต้องการ
2เปิด rustdesk.exe ในโฟลเดอร์
3ถ้า SmartScreen เตือน: More info → Run anyway
4แอปเชื่อมเซิร์ฟเวอร์ ZenityX อัตโนมัติ — พร้อมใช้
คำเตือนเกิดเพราะยังไม่ได้เซ็นใบรับรอง (กำลังดำเนินการ) — ไฟล์ปลอดภัย
