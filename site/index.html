วิธีติดตั้ง

ครั้งแรกอาจเจอคำเตือนความปลอดภัยของระบบ — ทำตามขั้นตอนนี้

🍎 macOS

  1. เปิดไฟล์ .dmg แล้วลาก ZenityX Desk ลงโฟลเดอร์ Applications
  2. เปิดครั้งแรก: คลิกขวา ที่แอป → Open → ยืนยัน Open
  3. ไปที่ System Settings → Privacy & Security → เปิด Screen Recording + Accessibility ให้ ZenityX Desk
  4. ปิดแล้วเปิดแอปใหม่ (จำเป็นสำหรับ Screen Recording)
ถ้าขึ้น "unidentified developer" ให้ใช้คลิกขวา → Open ครั้งแรกเท่านั้น

🪟 Windows

  1. แตกไฟล์ .zip ลงโฟลเดอร์ที่ต้องการ
  2. เปิด rustdesk.exe ในโฟลเดอร์
  3. ถ้า SmartScreen เตือน: คลิก More info → Run anyway
  4. โปรแกรมจะเชื่อมต่อเซิร์ฟเวอร์ ZenityX อัตโนมัติ
